
Programación funcional en Javascript
====================================

Este cursillo consta de un conjunto de ejercicios interactivos para aprender a programar con la biblioteca Javascript Rx (Reactive Extensions) de Microsoft. ¿Por qué entonces se llama "Programación funcional en Javascript"? Bueno, es que la clave para aprender Rx es saber utilizar programación funcional para operar con colecciones. La programación funcional proporciona a los desarrolladores las herramientas necesarias para abstraer como bloques que se pueden componer entre sí las operaciones habituales con colecciones. Te sorprenderás al aprender que la mayoría de las operaciones que realizas sobre una colección se pueden llevar a cabo con cinco sencillas funciones:

1. map
2. filter
3. mergeAll
4. reduce
5. zip

Te prometo los siguiente: si aprendes estas 5 funciones tu código será más corto, más auto descriptivo, y más mantenible. También, por razones que ahora no te resultarán obvias, aprenderás que estas cinco funciones son la clave para simplificar la programación asíncrona. Una vez hayas terminado este tutorial también tendrás todas las herramientas que necesitas para evitar fácilmente condiciones de carreras, para propagar y manejar errores asíncronos, y para secuencializar eventos y peticiones AJAX. Resumiendo, *estas 5 funciones serán probablemente las más potentes, flexibles y útiles que aprenderás jamás.*

Comencemos ahora con este tutorial [on-line]("http://pheras.github.com/aprenderx/").
